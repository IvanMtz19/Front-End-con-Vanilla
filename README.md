### 📝 Descripción del Proyecto
Esta es una Single Page Application (SPA) construida con **Vanilla JavaScript** y empaquetada utilizando **Vite**. 

El objetivo principal de esta aplicación es consumir datos externos desde la API pública `JSONPlaceholder` utilizando la API nativa `fetch()` y Promesas de JavaScript.

### ⚙️ Características y Opciones del Menú
* **Home:** Pantalla de bienvenida al usuario.
* **Posts:** Tabla interactiva que muestra las publicaciones obtenidas del endpoint `/posts`.
* **Todos:** Lista de tareas obtenidas del endpoint `/todos`, resaltando visualmente las tareas completadas y pendientes.
* **Usuarios:** Tabla con la información de contacto de los usuarios obtenida del endpoint `/users`.
* **About:** Información general sobre el propósito del proyecto.
